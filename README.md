# Ten-o/Pan123_Downloader

## 项目简介

> 123某盘下载器  模拟安卓客户端去下载 可能会造成封号。

## 目录结构

```bash
Pan123_Downloader/           # 根目录
├── Pan123_Downloader.py     # 主函数
├── users.json               # 用户信息
├── tokens.json              # 缓存token
└── README.md                # 项目说明文档
```

## 使用说明

### 1. 运行

```bash
python Pan123_Downloader.py
```
